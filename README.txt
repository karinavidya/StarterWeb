Initial Workspace Setup
1.  Go to https://github.com/karinavidya/StarterWeb
2.  Hover over and click the logo that says "Save karinavidya/StarterWeb to your computer and use it in GitHub Desktop."
3.  Given GitHub Desktop has already been installed, verify that step 2 opens GitHub Desktop program by default
4.  Manually adjust the repository configuration and path setup as shown on screen according to personal preferences
5.  Once the repository has been set up, right-click on StarterWeb repository and choose the option Open in Explorer
6.  For layout viewing, once the window has been opened, drag the main_content.php file to any web browsers
